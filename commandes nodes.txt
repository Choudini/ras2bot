Quand le bot est lancé avec la fenetre node : 

//éteindre le bot 

pm2 stop 0 

//allumer le bot 

pm2 start 0 

//redemarrer le bot 

pm2 restart 0 

//voir les messages d'erreur (100 derniers messages)

pm2 log main 0 --lines 100


Si il y a un problème lance help.exe.
